# Changelog

## 0.1.0

- Initial local package scaffold.
- Added core ScriptableObject data models.
- Added first-pass EditorWindow, asset creation, and validation flow.
- Added repair Timeline generation with visible animation clips.

- Added avatar frame and dialogue background configuration per scene and per dialogue line.
- Added editor dialogue preview using the configured StoryTalkPanel UI.
- Added scrollable dialogue history preview that appends lines and closes when dialogue ends.
- Added Restaurant demo generation with StoryTalkPanel UI assets.
